## A propos
Les components popup seront les components modaux utilisés dans l'application pour afficher des informations supplémentaires ou des formulaires sans quitter la page actuelle. Ils sont conçus pour améliorer l'expérience utilisateur en fournissant des interactions contextuelles et fluides.

## Usage
Faites seulement les components dans un nouveau dossier sous `src/app/planning/components/popup-components/` et exportez-les par défaut. Vous pouvez ensuite les importer et les utiliser dans d'autres parties de l'application selon vos besoins. 

Un component modal sera créer pour encadrer les différents components popup et gérer leur affichage et leur fermeture de manière cohérente.
